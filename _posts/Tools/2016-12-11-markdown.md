使用 Markdown[^1]可以效率的书写文档, 直接转换成 HTML[^2], 你可以使用 Leanote[^Le] 编辑器进行书写。它的目标是实现易读易写，成为一种适用于网络的书写语言。同时，Markdown支持嵌入`html`标签。

# Markdown的常用语法

## 标题
在行首加`#`表示不同级别的标题(H1-H6)
```text
# 一级标题
## 二级标题
### 三级标题
```
**演示**
# 一级标题
## 二级标题
### 三级标题
 
## 字体

使用 `*` 、 `**`、`***`和`~~` 分别表示斜体、粗体、斜粗体和删除线。
```text
*斜体*
**粗体**
***斜粗体***
***删除线***
```
**演示**
* *斜体*
* **粗体**
* ***斜粗体***
* ~~删除线~~

## 列表
常见列表的有序、无序列表，定义型列表，还可以将列表嵌套起来形成嵌套列表。

### 无序列表
使用 `*`，`+`，`-`后跟空格表示无序列表。
```text
* 无序一
* 无序二
```
**演示**
* 无序一
* 无序二

###  有序列表
使用数字和点后跟空格表示无序列表。
```text
1. 有序一
2. 有序二
```
**演示**
1. 有序一
2. 有序二

###  定义型列表
定义型列表由名词和解释组成。一行写上定义，紧跟一行写上解释。解释的写法:紧跟一个缩进(Tab)
```text
Markdown
:    轻量级文本标记语言，可以转换成html，pdf等格式
```
**演示**
Markdown
:    轻量级文本标记语言，可以转换成html，pdf等格式

###  嵌套列表
第一级列表用`*`+空格+内容，回车换行后，`Tab`+`＋或者－`+内容
```text
* 无序1
    + 无序1.1
    + 无序1.2
* 无序2
```
**演示**
* 无序1
    + 无序1.1
    + 无序1.2
* 无序2

## 文字引用
使用 > 表示文字引用。其他一些 Markdown 方言(例如` Mou `)的语法，换行需要在行末插入两个以上空格
```text
>朝辞白帝彩云间  
>千里江陵一日还  
>两岸猿声啼不住  
>轻舟已过万重山  
```

**演示**
>朝辞白帝彩云间  
>千里江陵一日还  
>两岸猿声啼不住  
>轻舟已过万重山  

## 表格    
表格的列于列之间用`|`隔开，第一行为表头，第二行分隔表头和主体部分,它可以为不同的列指定对齐方向，第三行开始每一行为一个表格行。
```text
|dog|bird|cat|
|:-:|:-:|:-:|
|foo|foo|foo|
|bar|bar|bar|
```
**演示**

|dog|bird|cat|
|:-:|:-:|:-:|
|foo|foo|foo|
|bar|bar|bar|

## 公式
行内公式用`$`整行公式用`$$`
```text
$E=mc^2$
$$x = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$
```
**演示**

行内公式$E=mc^2$    
整行\$$x = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$

## 流程图
将流程图代码包含在```folw和```之间即可，流程图的语法大体分为两部分
1. 前面部分用来定义流程图元素
2. 后面部分用来连接流程图元素，指定流程图的执行走向

```text
```flow
st=>start: 开始
e=>end: 结束
op=>operation: 操作
sub=>subroutine: 子程序
cond=>condition: 是或者不是?
io=>inputoutput: 输出

st(right)->op->cond
cond(yes)->io(right)->e
cond(no)->sub(right)->op
​```
```

## 分隔线
可以在一行中`***`、`---`、`___`来建立一个分隔线
```text
***
```

## 链接
链接包括内链接和外链接，实现当前页面和页间的跳转。
###  内链接
内链接其实就是锚点，也就是链接本文档内部的某些元素，实现当前页面中的跳转。在你准备跳转到的指定标题后插入锚点{#标记}，然后在文档的其它地方写上连接到锚点的链接。
```text
# Markdown的常用语法
跳转到[markdown的常用语法](#markdown的常用语法)
```
**演示**

跳转到[markdown的常用语法](#markdown的常用语法)

###  外链接
使用\[描述](链接地址) 为文字增加超链接
```text
[gary博客](http://blog.zhouming.work)
```
**演示**

欢迎来到[gary博客](http://blog.zhouming.work)

## 插入图像
图片与链接的语法很像，区别在一个!号
```text
![图片Alt](地址 Title)
```

## 换行
在行尾添加两个空格加回车表示换行
```text
我爱你  
中国
```

**演示**

我爱你  
中国
