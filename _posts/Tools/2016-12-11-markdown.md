使用 Markdown[^1]可以效率的书写文档, 直接转换成 HTML[^2], 你可以使用 Leanote[^Le] 编辑器进行书写。它的目标是实现易读易写，成为一种适用于网络的书写语言。同时，Markdown支持嵌入`html`标签。

# Markdown的常用语法

## 标题
在**标题开头** 加上1~6个`#`，依次代表一级-六级标题
```text
# 一级标题
## 二级标题
### 三级标题
```
**演示**
# 一级标题
## 二级标题
### 三级标题
 



## 字体

使用 `*` 、 `**`、`***`和`~~` 分别表示斜体、粗体、斜粗体和删除线。
```text
*斜体*
**粗体**
***斜粗体***
***删除线***
```
**演示**
* *斜体*
* **粗体**
* ***斜粗体***
* ~~删除线~~

## 列表
常见列表的有序和无序列表。

### 无序列表
使用 `*`，`+`，`-`后跟空格表示无序列表。
```text
* 无序一
* 无序二
```
**演示**
* 无序一
* 无序二

###  有序列表
使用数字和点后跟空格表示无序列表。
```text
1. 有序一
2. 有序二
```
**演示**
1. 有序一
2. 有序二

###  嵌套列表
第一级列表用`*`+空格+内容，回车换行后，`Tab`+`＋或者－`+内容
```text
* 无序1
    + 无序1.1
    + 无序1.2
* 无序2
```
**演示**
* 无序1
    + 无序1.1
    + 无序1.2
* 无序2

###  定义型列表
定义型列表由名词和解释组成。一行写上定义，紧跟一行写上解释。解释的写法:紧跟一个缩进(Tab)
```text
Markdown
:    轻量级文本标记语言，可以转换成html，pdf等格式
```
**演示**
Markdown
:    轻量级文本标记语言，可以转换成html，pdf等格式

## 文字引用
使用 > 表示文字引用。其他一些 Markdown 方言(例如` Mou `)的语法，换行需要在行末插入两个以上空格
```text
>朝辞白帝彩云间  
>千里江陵一日还  
>两岸猿声啼不住  
>轻舟已过万重山  
```

**演示**
>朝辞白帝彩云间  
>千里江陵一日还  
>两岸猿声啼不住  
>轻舟已过万重山
