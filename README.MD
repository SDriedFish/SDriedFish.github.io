#### 使用
1. fork库到自己的github
2. 修改名字为：username.github.io
3. 设置 _config.yml，将需要添加的社交帐号填入对应设置
4. 如果像我一样改掉```username.github.io```,只要在这CNAME中写入域名就可以了。不过你需要去域名服务商那里设定域名[解析规则][2]。
5. 在_posts中的目录结构添加博客文件即可
